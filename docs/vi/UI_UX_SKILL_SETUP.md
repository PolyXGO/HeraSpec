# Hướng Dẫn Thiết Lập UI/UX Skill

> **Lưu ý**: Tài liệu này đang được dịch. Hiện tại có thể xem bản tiếng Anh đầy đủ tại [UI_UX_SKILL_SETUP.md](../en/UI_UX_SKILL_SETUP.md)

## Tổng Quan

Hướng dẫn cách thiết lập và sử dụng UI/UX skill trong HeraSpec.

UI/UX skill tích hợp với UI/UX Builder search engine (được phát triển thêm từ base của [UI UX Pro Max Skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)).

Tài liệu đầy đủ bằng tiếng Việt đang được phát triển và sẽ sớm có sẵn.
